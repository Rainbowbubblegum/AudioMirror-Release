# ShareAudio - Download

**Mirror your headset audio to speakers** — Simple desktop app to route Windows audio to multiple output devices at once, with per-device volume controls.

## Download

**[Download Audio Mirror v1.0.0](https://github.com/Rainbowbubblegum/AudioMirror-Download/releases/latest)** (Windows 10/11, ~50 MB)

## Installation
You can run **AudioMirror.Desktop.exe** directly from the extracted folder — no install needed (portable mode).

## What it does

- Mirror audio from your default playback device (e.g. headphones) to other devices (e.g. speakers)
- Control volume for each output device independently
- Start/stop with one click
- Real-time audio level meter
- Windows 10 & 11 compatible

## Requirements

- Windows 10 or 11
- No additional software needed (includes everything)
